# Asset Prompts — Wafinix

Prompt siap-pakai untuk image generator (Midjourney / Flux / Ideogram / DALL·E /
Imagen / Gemini-nano-banana). Disusun dari prinsip skill `brandkit` &
`imagegen-frontend-web`, dikunci ke brand Wafinix.

**Cara pakai:** untuk aset ber-brand Wafinix (§1–§3, §5), tempelkan **Style DNA (§0)**
di awal/akhir prompt agar palet konsisten. Untuk mockup portofolio (§4), JANGAN pakai
Style DNA — tiap proyek itu brand fiktif berbeda (punya palet sendiri).

**Tips tool:** UI/mockup dengan teks terbaca → Ideogram atau Flux (atau nano-banana
setelah billing). Ilustrasi/brand board → semua tool oke. Untuk teks tajam, ketik
teksnya eksplisit di prompt & pakai tool yang kuat tipografi (Ideogram).

## §0 — Style DNA (preamble brand Wafinix — tempel ke §1–§3, §5)

```
Wafinix brand visual identity: a warm, human, approachable software house.
Color palette — background cream #FAF6F0, primary terracotta #C8623E, accent soft
amber #E3A547, deep espresso brown #33261C for text and dark surfaces, muted sage
green #8A9B7C as tertiary. Mood: warm, premium, modern, editorial, calm, trustworthy.
Visual language: flat 2D vector illustration with a subtle paper-grain texture,
rounded organic shapes, soft warm directional light, generous negative space, refined
minimal composition. Brand motif: a stylized phoenix and an upward-growth gesture.
```
**Negative / avoid (append where supported):**
```
no readable gibberish text, no watermark, no other brand logos, no cold corporate
blue, no neon, no harsh gradients, no clutter, no generic stock-photo people, not busy,
no 3D glossy plastic look.
```

## §5 — Spot illustration template (per layanan / industri)

Gaya seragam, dipakai untuk kartu layanan/industri bila ingin mengganti ikon lucide.
Isi `<subjek>` lalu tempel Style DNA §0.

```
A small flat 2D vector spot illustration of <subjek>, centered with generous padding,
rounded organic shapes, 2–3 colors from the palette only, subtle grain, soft warm
light, simple and iconic, transparent or cream background, no text. [+ Style DNA §0]
```
Contoh `<subjek>`: layanan → "a browser window with a rising growth arrow" (Website),
"a cash register / receipt" (POS), "a credit card with a checkmark" (Payments),
"connected modular blocks" (ERP), "a friendly chat bubble with a spark" (AI & Chatbot),
"a smartphone app" (Mobile), "a cursor sprinkling sparkles" (UI/UX & Motion).
Industri → "a coffee cup / market stall" (UMKM), "a cargo ship" (Ekspor-Impor),
"a dining plate" (Restoran), "a hotel bell" (Hotel), "a palm beach" (Pariwisata),
"an airplane ticket" (Web Trip), "a house with a tag" (Properti), "a Kaaba silhouette"
(Haji & Umroh), "an office building" (Korporat).
